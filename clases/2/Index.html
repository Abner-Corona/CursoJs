Origen

El nombre HTML proviene de las siglas en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto), el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de “hipertexto” para compartir documentos. El HTML se hizo popular por su sencillez, era fácil de aprender y eso lo hace accesible a mayor número de personas. Estos documentos web deben estar escritos con el mismo “lenguaje” para que diferentes ordenadores puedan leerlos, si alguien utiliza un sistema diferente no podrá compartir su información con los que usan el HTML ni podrá visualizar los de los demás.

Estructura

En el interior de las etiquetas <head></head> se escribirán las líneas de código que definan el propio documento (Nombre, idioma, propósito de la web, llamadas a hojas de estilo y otros lenguajes…).
En el interior de las etiquetas <body></body> se escribirán las líneas que definan su estructura y contenido: Tablas, párrafos, imágenes, vídeos…
