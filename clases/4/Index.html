Variables

Una variable es un elemento que se emplea para almacenar valores. Las variables en JavaScript se crean mediante la palabra reservada var, solamente se debe indicar al definir por primera vez la variable, lo que se denomina declarar una variable.

Tipo de dato numérico

Se utilizan para almacenar valores numéricos enteros (llamados integer en inglés) o decimales (llamados float en inglés).
